# genome_diagram
Use of GenomeDiagram for phage genome visualisation and annotation



# Folder structure

The program will automatically create the following folder architecture in the provided directory:

.
|-- genbank
|-- genome_download
|-- processed_files
|   |-- fasta
|   |-- blastn_database
|   |-- blastn
|-- tables
|   |-- blastn_summary
|   |-- locus_and_gene
|   |-- gene_presence
|--synteny
|   |-- gene_uniqueness
|   |-- graph
